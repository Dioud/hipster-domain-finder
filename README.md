# Hipster Domain Finder

Find OG domain hacks with [Hipster Domain
Finder](http://www.hipsterdomainfinder.com).

Current hacks:

- er -> r (ex: tumbler -> tumblr.com)

- tld in word (ex: lobsters -> lobste.rs)
